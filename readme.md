## 个人简历模板

缘由： 网上找了半天找的眼睛花了，也没找到好看的简历模板，一怒之下自己写了一个。

此网页最初来自于Typora导出的HTML格式，自己在其中加入了一些markdown无法实现的功能（浮动靠右、li两列显示、图片位置调整等）



字体使用的是`阿里巴巴普惠体`，[下载地址](https://ics.alibaba.com/project/Hn8mXx)
